This page highlights several research projects related to Artificial Intelligence, Data Science, Natural Language Processing, and Educational Technology.

#### 1. Automating Systematic Literature Reviews (SLR) with LangChain and Open-Source LLMs
This project focuses on automating key stages of the Systematic Literature Review (SLR) process using LangChain and open-source Large Language Models (LLMs). The system assists researchers in literature search, paper screening, information extraction, study classification, and evidence synthesis, reducing the time and effort required for manual review.

By leveraging LangChain for workflow orchestration and open-source LLMs for natural language understanding, the platform can process large collections of research articles, identify relevant studies based on predefined criteria, extract important findings, and generate structured summaries. The project aims to improve the efficiency, scalability, and reproducibility of SLRs while maintaining transparency through customizable and explainable AI-driven pipelines.

#### 2. Anything-to-Problem Generation with Generative AI
This project develops an AI-powered system that automatically generates educational and assessment problems from various input sources, including text, documents, articles, images, and structured data. Using LangChain and open-source Large Language Models (LLMs), the system transforms content into diverse question formats such as multiple-choice, short-answer, and problem-solving tasks.

The platform supports customizable difficulty levels, learning objectives, and domain-specific requirements, enabling educators to create high-quality assessment materials efficiently. The project aims to reduce question authoring effort while improving the scalability and accessibility of educational content generation.

#### 3. Code Error Interpreter Chatbot with Fine-Tuned Open-Source LLMs
This project develops an AI-powered chatbot that helps programmers understand, diagnose, and resolve coding errors using fine-tuned open-source Large Language Models (LLMs). The model is trained on programming-related datasets, error logs, debugging examples, and technical documentation to improve its ability to interpret and explain code errors accurately.

Supporting multiple programming languages and development environments, the chatbot analyzes error messages, identifies root causes, and provides context-aware debugging guidance and code correction suggestions. The project aims to enhance developer productivity and learning by delivering fast, reliable, and domain-specific assistance for software development and troubleshooting.

#### 4. Multimodal AI Agent for Language Learning
This project develops a multimodal AI agent that supports language learning through text, speech, images, and interactive conversations. Powered by open-source Large Language Models (LLMs) and multimodal AI technologies, the system provides personalized learning experiences, including vocabulary acquisition, grammar correction, pronunciation feedback, reading comprehension, and conversational practice.

The agent integrates multiple input modalities to create engaging and context-aware learning activities, enabling learners to interact naturally using voice, text, or visual content. The project aims to enhance language proficiency, learner engagement, and accessibility through adaptive and intelligent AI-assisted learning support.
