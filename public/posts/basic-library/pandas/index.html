1. What is Pandas?#

Pandas is an open-source Python library designed for data manipulation and analysis. It provides two powerful data structures:

  • Series, which stores one-dimensional labeled data.
  • DataFrame, which stores two-dimensional tabular data similar to a spreadsheet or SQL table.

Pandas simplifies tasks such as cleaning datasets, handling missing values, filtering records, computing statistics, and preparing data for visualization and machine learning. Built on top of NumPy, it combines high performance with an intuitive interface, making it one of the most widely used libraries in data science and analytics.

A. Installing Pandas#

Install Pandas using pip.

pip install pandas

B. Importing Pandas#

By convention, Pandas is imported using the alias pd.

import pandas as pd

2. Creating a Series#

A Series is a one-dimensional labeled array.

import pandas as pd

ages = pd.Series([20, 22, 21, 23])

print(ages)

Output

0    20
1    22
2    21
3    23
dtype: int64

3. Creating a DataFrame#

A DataFrame is a two-dimensional table consisting of rows and columns.

import pandas as pd

students = {
    "Name": ["Alice", "Bob", "Charlie"],
    "Age": [20, 21, 22],
    "Score": [85, 90, 88]
}

df = pd.DataFrame(students)

print(df)

Output

      Name  Age  Score
0    Alice   20     85
1      Bob   21     90
2  Charlie   22     88

B. Selecting Columns#

Select a single column.

print(df["Name"])

Output

1        Bob
2    Charlie
Name: Name, dtype: object

Select multiple columns.

print(df[["Name", "Score"]])

Output

      Name  Score
0    Alice     85
1      Bob     90
2  Charlie     88

D. Filtering Data#

Select students with scores greater than 85.

high_scores = df[df["Score"] > 85]

print(high_scores)

Output

      Name  Age  Score
1      Bob   21     90
2  Charlie   22     88

E. Adding a New Column#

df["Passed"] = df["Score"] >= 80

print(df)

Output

      Name  Age  Score  Passed
0    Alice   20     85    True
1      Bob   21     90    True
2  Charlie   22     88    True

F. Modifying Data#

df.loc[0, "Score"] = 95

print(df)

Output

      Name  Age  Score
0    Alice   20     95
1      Bob   21     90
2  Charlie   22     88

G. Sorting Data#

Sort by score in ascending order.

print(df.sort_values("Score"))

Output

      Name  Age  Score
2  Charlie   22     88
1      Bob   21     90
0    Alice   20     95

Sort in descending order.

print(df.sort_values("Score", ascending=False))

Output

      Name  Age  Score
0    Alice   20     95
1      Bob   21     90
2  Charlie   22     88

I. Handling Missing Values#

students = {
    "Name": ["Alice", "Bob", "Charlie"],
    "Score": [85, None, 90]
}

df = pd.DataFrame(students)

print(df)

Output

      Name  Score
0    Alice   85.0
1      Bob    NaN
2  Charlie   90.0

Check missing values.

print(df.isnull())

Output

    Name  Score
0  False  False
1  False   True
2  False  False

Fill missing values.

df["Score"] = df["Score"].fillna(0)

print(df)

Output

      Name  Score
0    Alice   85.0
1      Bob    0.0
2  Charlie   90.0

J. Grouping Data#

students = {
    "Department": ["IT", "IT", "CS", "CS"],
    "Score": [80, 90, 75, 85]
}

df = pd.DataFrame(students)

print(df.groupby("Department")["Score"].mean())

Output

Department
CS    80.0
IT    85.0
Name: Score, dtype: float64

K. Saving Data#

Save a DataFrame as a CSV file.

df.to_csv("students.csv", index=False)

4. Practical Example#

Calculate the average score of students who passed.

import pandas as pd

students = {
    "Name": ["Alice", "Bob", "Charlie", "David"],
    "Score": [85, 92, 76, 88]
}

df = pd.DataFrame(students)

passed = df[df["Score"] >= 80]

print(passed)

print("\nAverage Score:", passed["Score"].mean())

Output

    Name  Score
0  Alice     85
1    Bob     92
3  David     88

Average Score: 88.33333333333333

5. Common Errors#

A. Forgetting to Import Pandas#

df = pd.DataFrame()

Error

NameError: name 'pd' is not defined

Solution

import pandas as pd

B. Accessing a Non-Existent Column#

print(df["Grade"])

Error

KeyError: 'Grade'

Ensure the column name exists before accessing it.

C. Incorrect Column Name#

print(df["score"])

Error

KeyError: 'score'

Remember that column names are case-sensitive.
